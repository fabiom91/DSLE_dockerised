### Setup
1. Create a virtual environment with Python 3 (>= 3.6 recommended)
2. Run
```
pip install -r requirements.txt
```
3. Create a new database
```
db.create_all() # TODO: Expand on this
```
4. Do some additional setting up
5. Launch server
```
python app.py
```
